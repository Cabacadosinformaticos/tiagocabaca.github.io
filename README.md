# 🌐 Personal Website

## 📌 Overview

This repository contains the development of my personal website and portfolio.

The purpose of this project is to create a centralized digital space where I can present my projects, technical skills, academic work, and ongoing development in the field of software engineering and technology.

---

## 🎯 Objectives

* Build a professional personal portfolio
* Showcase academic and personal projects
* Create a platform for continuous updates and experimentation

---

## 📈 Personal Development Goals

* Improve front-end development skills
* Experiment with UI/UX design
* Explore modern web technologies and best practices
* Deploy a real-world web project  

---

## 🚀 Features

* Fully responsive design (desktop, tablet, mobile)
* Dedicated sections:

  * Home
  * About Me
  * Projects / Portfolio
  * Contact
* Clean and modern user interface
* Easy navigation and structured layout
* Scalable architecture for future improvements

---

## 🛠️ Technologies Used
